First release of **AegisMac** — a native Swift/SwiftUI port of [Aegis Authenticator](https://github.com/beemdevelopment/Aegis) for macOS. Vault files are byte-compatible with the Android app: the same `aegis.json` moves freely between devices.

> This is an unofficial port, not associated with or endorsed by Beem Development. It is also **vibe coded** — written almost entirely by Claude (Anthropic's Fable model) from behavior specs and test vectors extracted from the original Java source, backed by 140 unit tests (RFC 4226/6238 vectors, upstream test data, encrypted-vault round-trips against Android-produced fixtures) and regular syncing with upstream. Audit accordingly, and keep backups of your vault.

## Highlights

- **OTP algorithms**: TOTP (RFC 6238), HOTP (RFC 4226), Steam, MOTP, Yandex — SHA1/SHA256/SHA512, 6–10 digits
- **Vault compatibility**: reads and writes the Android Aegis vault format (envelope v1, db v3), encrypted (scrypt + AES-256-GCM) or plaintext
- **Touch ID** unlock (password stored in the login keychain behind biometric access control)
- **Import**: `otpauth://` URIs, Google Authenticator migration QR codes, Aegis JSON exports; QR detection from image files or directly off the screen
- **Export**: encrypted or plaintext Aegis JSON, `otpauth://` URI list
- **UI faithful to the original**: Aegis Material palette, Light/Dark/AMOLED themes, favorites, groups, search, four view modes — adapted to macOS conventions

## Install

Requires macOS 14 (Sonoma) or later, Apple Silicon or Intel.

1. Download `AegisMac-0.1.0.dmg` below and drag **AegisMac** to Applications.
2. The app is ad-hoc signed, not notarized, so macOS blocks the first launch. Allow it under **System Settings → Privacy & Security → Open Anyway**, or clear the quarantine flag:

   ```sh
   xattr -d com.apple.quarantine /Applications/AegisMac.app
   ```

3. Optionally verify the download:

   ```sh
   shasum -a 256 -c SHA256SUMS.txt
   ```

Your vault lives at `~/Library/Application Support/AegisMac/aegis.json` — a standard Aegis vault the Android app can import directly. Back it up.

## Known limitations

- Ad-hoc signed only (no Developer ID / notarization), hence the first-launch hoop above.
- No auto-update — watch this repository's Releases for new versions.
- Icon packs and automatic backups from the Android app are not implemented.

Based on upstream Aegis commit `a9d45b3` (after v3.4.2). Issues and feedback welcome on the [issue tracker](https://github.com/myl7/AegisMac/issues).
